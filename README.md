# Credit Card Fraud Detection using Machine Learning

## Overview

This project focuses on detecting fraudulent credit card transactions using Machine Learning techniques. The objective is to build and evaluate multiple classification models capable of distinguishing between legitimate and fraudulent transactions in a highly imbalanced dataset.

## Dataset

The project uses the **Credit Card Fraud Detection Dataset** from Kaggle.

### Dataset Information

* Total Transactions: 284,807
* Fraudulent Transactions: 492
* Legitimate Transactions: 284,315
* Features: 30 numerical features
* Target Variable:

  * 0 = Legitimate Transaction
  * 1 = Fraudulent Transaction

## Project Objectives

* Analyze and understand transaction patterns.
* Handle class imbalance in the dataset.
* Train multiple classification models.
* Compare model performance using standard evaluation metrics.
* Identify the most effective model for fraud detection.

## Data Preprocessing

* Data exploration and visualization
* Feature scaling
* Handling class imbalance
* Train-test split
* Data preparation for machine learning models

## Machine Learning Models

The following supervised learning algorithms were implemented:

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Tree
* Random Forest

## Evaluation Metrics

Models were evaluated using:

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix
* ROC Curve
* AUC Score

## Results

The models were compared using a standardized evaluation framework.

Key findings:

* Random Forest achieved the best overall performance.
* Decision Tree provided better interpretability.
* Recall and F1-Score were prioritized due to the highly imbalanced nature of the dataset.
